# LibSearch Component

This component provides a search functionality for the library. It will allow users to search through various components and documentation available in the library.

Editing the components content is done through the `main.mjs` file located in the same directory.

## File Structure
- `main.mjs`: Contains the main logic and structure for the LibSearch component.
- `README.md`: This documentation file.
- `preview.html`: A preview file to visualise the component during development.

## To Run Locally

1. Clone the repository to your local machine.
2. Navigate to the `src/global_components/content/LibComponents/LibSearch/` directory.
3. Open a terminal and ensure that the correct directory is open
4. Run the following command to start the deployment `dxp-next cmp dev-ui <component-name>`
5. Open your web browser and go to `http://localhost:3000` to view the component in action.
